# 小六壬起卦 | Xiao Liu Ren Divination

> 基于传统小六壬（诸葛马前课）的在线起卦工具，支持三宫综合解读、分类断事、五行生克分析。纯前端实现，无需后端，可直接部署到 GitHub Pages。

🔗 **在线体验**：[网页链接](https://fyapeng.com/xliuren)

---

## ✨ 功能特性

### 起卦方式
- **按当前时间**：自动获取公历时间，转换为农历月、日、时辰进行起卦
- **按随机数字**：手动输入三个数字起卦，适用于心中默念随机数的场景

### 三宫综合解读
不同于只看最终落宫的简单版本，本工具展示完整的三宫信息：
- **第一宫（月上）**：事情的起因与背景
- **第二宫（日上）**：事情的经过与发展
- **第三宫（时上）**：最终结果与现状

### 分类断事
支持 8 种事项的针对性解读：
- 综合 · 求财 · 婚恋 · 事业 · 出行 · 疾病 · 失物 · 诉讼

每个宫位在不同事项下有独立的吉凶判断和详细解读。

### 五行生克分析
- **三宫五行关系**：分析月宫、日宫、时宫之间的五行生克，判断起因对发展、发展对结果的影响
- **时辰与落宫生克**：分析当前时辰五行与最终落宫五行的关系（相生、相克、泄气、受制、比和），判断外部时机对事情的影响
- **六神对应**：青龙、玄武、朱雀、白虎、六合、勾陈

### 其他
- 农历转换使用浏览器原生 `Intl.DateTimeFormat` API，精度高且零依赖
- 移动端适配，三宫卡片保持横排对比
- 实时时钟显示公历、农历、时辰及五行

## 🔧 技术实现

### 农历转换

放弃了常见的 `lunarInfo` 十六进制查表方案（网上流传的数据表存在错误），改用浏览器原生 Intl API：

```javascript
const parts = new Intl.DateTimeFormat('zh-CN-u-ca-chinese', {
  year: 'numeric', month: 'long', day: 'numeric'
}).formatToParts(new Date(2026, 2, 27));
// → yearName: "丙午", month: "二月", day: "9"
```

优势：
- 零依赖，不需要任何第三方库
- 精度由浏览器内置 ICU 数据保证
- 支持闰月识别
- 兼容所有现代浏览器（Chrome 80+、Firefox 78+、Safari 14+）

### 起卦算法

```
月上宫 = (农历月 - 1) % 6
日上宫 = (月上宫 + 农历日 - 1) % 6
时上宫 = (日上宫 + 时辰序号 - 1) % 6
```

六宫顺序：大安(0) → 留连(1) → 速喜(2) → 赤口(3) → 小吉(4) → 空亡(5)

### 时辰五行

| 时辰 | 子 | 丑 | 寅 | 卯 | 辰 | 巳 | 午 | 未 | 申 | 酉 | 戌 | 亥 |
|------|---|---|---|---|---|---|---|---|---|---|---|---|
| 五行 | 水 | 土 | 木 | 木 | 土 | 火 | 火 | 土 | 金 | 金 | 土 | 水 |

与落宫五行的六种关系：相生（时生宫）、相克（时克宫）、泄气（宫生时）、受制（宫克时）、比和、无直接关系。

## 📖 小六壬简介

小六壬是中国民间流传的一种简易占卜方法，以大安、留连、速喜、赤口、小吉、空亡六宫按月日时顺数来判断吉凶，因其简单快速、掐指即可完成而广为流传。民间也有人将其与"诸葛马前课"混称，但严格来说二者是不同的体系——诸葛马前课（马前神课）另有一套以十二地支配合的断法，只是都强调行军途中快速占算这一特点，所以常被混用。

六宫分别为：

| 宫位 | 五行 | 六神 | 吉凶 |
|------|------|------|------|
| 大安 | 木 | 青龙 | 大吉 |
| 留连 | 水 | 玄武 | 凶带小吉 |
| 速喜 | 火 | 朱雀 | 吉 |
| 赤口 | 金 | 白虎 | 凶 |
| 小吉 | 水 | 六合 | 吉 |
| 空亡 | 土 | 勾陈 | 凶 |
